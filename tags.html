#orgmode

Org mode is an Emacs package for keeping notes, maintaining TODO lists, planning projects, and authoring documents with a fast and effective plain-text system. It's a very popular package among Emacs users, that can serve as foundational layer for thinking about knowledge storage and management.

related: #emacs

#spacemacs

Spacemacs is a community-driven Emacs distribution that can basically be applied on top of GNU Emacs and focuses on ergonomics, mnemonics, consistency, and discoverability by featuring an intuitive keybinging menuing system driven predominantly by keybindings that begin with the SPC (spacebar), versus unwieldy Emacs commands which often focus on C (control key), and M (ESC key) leader key combinations. I'm currently using Spacemacs and still working on configuring it to behave how I want it to, but for the most it came out of the box in a pretty reasonable state to function as a Vim replacement from a keybinding perspective (on Spacemacs develop branch).
