Notes

Using DB Browser for SQLite (DB4S) visual, open source GUI.

  • ?PATTERN?
    • A percent symbol % in the LIKE pattern mathces any sequence of zero or more characters in the string.
    • An underscore symbol _ in the LIKE pattern matches any single character in the string.
  • Querying:
    • Mostly like normal SQL as far as I can tell so far.
    • SQLite CLI

Commands

# Open the sqlite3 Cli for a given db_filename database filename.
$> sqlite3 db_filename.sqlite3

-- See the help menu.
sqlite> .help

-- Exit the CLI.
sqlite> .exit 

-- List all tables or tables matching a pattern LIKE TABLE.
sqlite> .tables
sqlite> .tables TABLE

-- Show a table's schema e.g. table name="documents".
sqlite> .schema documents 

-- Set the query output mode. Favorite modes: line, list.
sqlite> .mode list
