I'm keeping this list here as a reminder for myself of all the Emacs and Spacemacs commands I'm learning and finding useful:

Important

  • C-g: Quit current command sequence. Update 4:10 pm April 10, 2020: Also bound C-g to exit evil insert mode while in Org mode. See below.
  • C-c: I mapped C-c to go back to normal-mode, from insert-mode in order to make Spacemacs behave just a bit more like Vim.
    • Update 4:06 pm April 10, 2020: I ended up remapping C-g to exit evil mode because C-c made Magit unusable. I detailed this update here: Adding Another Macro to Exit Evil Insert Mode. I'm currently only binding it in Org mode and mostly only using Org mode in Emacs at this point, so I'm not sure if this might lead to other problems down the road.
  • SPC SPC: Execute Emacs commands by name.
  • SPC q s: Quit Emacs and save all buffers.
  • SPC q r: Restart Emacs and resume layouts. This is the best!
    • Just needed to add a call to (server-start) in my spacemacs config file in order to trigger the Emacs server to restart at the same time.

Windows

  • SPC w V: Split window right and focus. This was one of the things I'd do most frequently in Vim via :vsp.

Workspaces

Workspaces seem to behave kind of like Vim tabs.

  • SPC l w: List workspaces.
  • SPC l w s: Create a new single Window workspace.
  • SPC l w #: Goto workspace #.
  • SPC l w p: Goto previous workspace.
  • g t: Goto next workspace. (Vim/Evil)
  • g T: Goto previous workspace. (Vim/Evil)

Files and Buffers

  • SPC f f: Find in files.
  • SPC f r: Find in recent files.
  • SPC f s: Save the current file.
  • SPC f R: Rename current buffer file.
  • SPC b d: Kill buffer.
  • SPC b b: List open buffers.
  • SPC SPC shell RET: Start an interactive shell.
    • C-x k: Kill the shell when done with it.

Org mode

  • SPC m i d: Insert drawer.
  • C-c C-,: Insert structured templates including begin_src and begin_export, among others.
  • <sTab: Typing the sequence < s Tab creates a begin_src block.

Dates

  • SPC m d t: Insert date timestamp. Only contains the date.
  • SPC u SPC m d t: Insert time and date timestamp.

Links

  • SPC a o l: Store a link to the current file.
  • SPC m i l: Insert a link. Suggests the most recently stored link first.

Motion

From Motion.

  • C-c C-p: Jump to previous headline.
  • C-c C-n: Jump to next headline.
  • (: Jump to previous sentence.
  • ): Jump to next sentence.
  • {: Jump to previous sentence.
  • }: Jump to next sentence.

Text

  • SPC m x b: Make selected text bold.
  • SPC m x c: Make selected text into code.
  • SPC m x h: Make selected text highlighted. Customization. I detailed how here.
  • SPC m x i: Make select text italic.
  • SPC m x s: Make selected text stricken through.
  • SPC m x u: Make selected text underlined.
  • SPC m x v: Make selected text verbatim.

Git / Magit

  • SPC g s: Magit status for repository of current file.

In Magit Status Buffer

C-c C-c c c~: Commit staged files. Stage files if no files staged.

  • , ,: Finish and save commit message.
  • , c: Cancel commit.
  • P u: Push to upstream.
