A listing and descriptions of some of the various software applications in my software toolbox that I use across computers, phones, and tablets. I'm mostly talking about software tools that help enable productivity and efficiency on a daily basis as I'm doing things like surfing the web, taking notes, collaborating, and writing code and developing software.

I've tried my best to choose the tools that I'm using and listing here carefully. Over time, I've discarded a lot of tools that added too much complexity or lacked value. I'm mainly focusing here on tools that I enjoy using and are core to my workflows and productivity. For example, while I use things like Microsoft Word or Excel when I need to, I'm not including them here because they're not my preferred tools and I try to use them as infrequently as possible. I'm also currently excluding everyday tools like cat, ps, awk, Finder, Activity Monitor, etc. becuase of how common they are.

If you're reading through this list and have any suggestions for other great tools you use that you think might be useful, please share - I'd love to hear from you!

Note: Unless otherwise specified, the majority of these tools are things I'm using on my MacBook Pro, late 2019 model currently running Catalina 10.15.4. As of April 2020.

macOS Apps

DB Browser for SQLite

DB Browser for SQLite (DB4S) is a high quality, visual, open source tool to create, design, and edit database files compatible with SQLite.

Divvy

Divvy helps you manage your desktop your macOS desktop. It allows you to quickly and efficiently ‚Äúdivvy up‚Äù your screen into exact portions. Divvy makes it really easy to tile your windows, and supports adding your own keyboard shorcuts to define where to place a window.

Dropbox

Dropbox helps you effortlessly sync your files across machines and back them up in the cloud.

I've been using Dropbox for quite some time, but have recently heard good things about Nextcloud and Syncthing. I just haven't found compelling enough reasons to try either of them yet.

Emacs

I'm just getting started with Emacs, currently using the Emacs Mac Port installed through Homebrew to get the best of GUI Emacs along with Emojis üëå. Getting Emojis working was definitely an exciting upgrade for me, especially because I'm lucky enough to have a Mac with the Touch Bar that makes finding and using Emojis much easier, along with the Spacemacs Emoji Layer + Cheat Sheet.

Evernote

I've been using Evernote to store my notes since around 2010. At the time of this writing (April 2020) I have 3,330 notes stored in Evernote. I'm currently working on transitioning parts of my note-taking workflow to Emacs.

Google Chrome

Google Chrome has been my web browser of choice ever since I started using shortly after it was released for macOS in 2009. Before Chrome I had some great times using Firefox for its Firebug developer toolkit which was pretty amazing at the time.

Web Apps

  • Gmail has been my mail client for years now.
  • Google Docs is my preferred document document collaboration toolkit. I love all of the commenting and sharing features for their simplicity, and use Docs, Sheets, and Slides heavily both for my personal and professional projects wherever I can. I can't believe that they've actually made a great PowerPoint style slide editor that runs in the browser. I never figured out PowerPoint, but in contrast, Slides was incredibly easy to pickup and get started with.

Extensions

  • Hypothes.is: Store annontations from arouhnd the web while researching and taking notes.
  • LastPass: Manage and secure all of my passwords across devices.
  • Vimium: Vim-like keyboard shortcuts in Google Chrome.

Other

  • Pinboard: Store all my web bookmarks. I have become pretty liberal with my bookmarking and currently have more than four thousand bookmarks dating back to October 2015. Prior to Pinboard I was using Pocket, and prior to that I was using Instapaper.

GrandPerspective

I use GrandPerspective any time I need to look at managing my files and disk space. It's a simple app that that graphically shows the disk usage within a file system using tree maps.

iTerm2

I use iTerm2 on macOS for all my work in the terminal. iTerm2 has a lot of great features, but I mainly use it because it looks nicer than the macOS terminal application.

Kindle

I've started using Kindle to read books again across devices now that I've found it has the ability to let me sync highlights across devices and download all my highlights as an HTML file to add to my notes.

LastPass

I'm using LastPass to help me manage and secure all of my passwords across devices.

Little Snitch

Little Snitch enables you to see all of your internet connections so that you can track who your computer is talking to online.

Lockdown

Lockdown is a simple, powerful firewall that blocks tracking, ads, badware and more. Free and open source. The authors built Lockdown for Privacy management but it's also pretty effective at speeding up web page loading by reducing advertisting and analtyics traffic.

Micro Snitch

Micro Snitch is an ultra-light menu bar application operates inconspicuously in the background. It monitors and reports any microphone and camera activity to help you figure out if someone‚Äôs spying on you.

Quicksilver

Quicksilver is a fast and free macOS productivity application that gives you the power to control your Mac quickly and elegantly. Quicksilver learns your habits, making your everyday chores simple and efficient.

I'm currently just using Quicksilver to map some a few shortcut keys to open or change too specific applications.

SequelPro

SequelPro is my goto MySQL GUI client. Although, I've heard good things about TablePlus and I'm looking forward to checking it out one of these days.

macOS Spotlight

I included Spotlight here because even thought it's pretty common, I wanted to highlight that it's still my finder of choice for simple and quick app navigation. Opening applications quickly by name is really about all that I do with it though.

Stay

Stay is another window management program that stores all your app window positions and ensures that your windows are where you left them as you disconnect and reconnect across different monitor configruations.

tmux

tmux is a terminal multiplexer. It enables a number of terminals to be created, accessed, and controlled from a single screen. tmux may be detached from a screen and continue running in the background, then later reattached. It runs on OpenBSD, FreeBSD, NetBSD, Linux, OS X and Solaris.

Vim (the ubiquitous text editor)

Vim is a highly configurable text editor for efficiently creating and changing any kind of text. It is included as "vi" with most UNIX systems and with Apple OS X.

Vim was my first love in editors, and I'm still using it for all my source code and terminal work as of April 2020, since going back to sometime in 2003.

Command Line and Utilities

Bash

Bash is the GNU Project's shell. Bash is the Bourne Again SHell. Bash is an sh-compatible shell that incorporates useful features from the Korn shell (ksh) and C shell (csh).

ctop

ctop is a top-like command for tracking container metrics.

curl

curl is a command line tool and library for transferring data with URLs.

glances

glances is a top/htop alternative for GNU/Linux, BSD, Mac OS and Windows operating systems.

htop

htop is an interactive text-mode process viewer for Unix systems. It aims to be a better 'top', and where I have it installed I prefer it's more graphical represenation of metrics like RAM and cores usage.

jq

jq is a lightweight and flexible command-line JSON processor. This is my goto tool for beautifying and inspecting JSON in the terminal.

Python

Any time I just want to do some simple calculations or write a quick function, Python is my usually my command line interpreter (cli) of choice (unless I happen to have a Rails Console nearby).

Rails Console

The Rails Console is one of the super useful parts of Ruby on Rails that enables doing a lot of code prototyping and testing of your application code.

Software Development

These are all simply my personal preferences. Usually these are driven by my own familiartiy and efficiency, as well as community size and support/ecosystem quality.

Development Environment

The majority of the recent projects that I've worked on have involved development processes built around Docker.

For application development, I prefer to work using Docker üê≥ and Compose üêô as much as possible to spin up and manage my local development stack.

Operating Systems

  • macOs: First choice for personal OS because of wider developer use than Linux. There just aren't enough great Linux tools that are quite as easy to use as OS X tools, imho.
  • Ubuntu: I had been using CentOS for Linux servers for quite some time, but have recently switched to Ubuntu given it's growing ubiquity.
  • Alpine: I always try to use Alpine Linux for container development for it's minimal footprint and approach.

Web Servers

  • Ruby on Rails: I've been using Ruby on Rails since something like version 1.3* back in 2007.
  • Flask: I had used Django in the past for more client facing web server work, but as I've focused on using Ruby on Rails for more client-side web serving and Python for backend APIs it has seemed that Flask has emerged as a strong candidate in the Python API server space for it's simplicity and community.

Other

Authy

Authy is my favorite app for managing as many of my Two Factor authentication integrations as possible.

Have I Been Pwned (HIBP)

HIBP notifies you when you have an account that's been compromised in a data breach. Being notified about data breaches and compromised account information is always a great reminder and motivator for maintaining good security hygiene, as well as being sure to change passwords, and delete old unused accounts.
